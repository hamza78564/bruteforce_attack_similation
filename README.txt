Brute Force Attack Simulation Project
## 🛠️ Project Components

The project consists of two main parts:
1. **The Server (Victim):** A simple local web application built using the Flask framework.
2. **The Attacker (Script):** A Python script that uses the **Requests** library to send multiple HTTP requests and test a password list (wordlist) to find the correct password.

---

Ensure Flask and Requests are installed on your system:
